多个单文件组件使用

在项目目录下下创建components文件夹，然后将所有子组件放入components文件夹下。

1. 多组件嵌套使用

1）编写Child1.vue文件的内容。

<template>
    <div>单文件组件：Child1</div>
</template>

<script>
    export default {

    }
</script>

<style scoped>

</style>

2）编写Child2.vue文件的内容。

<template>
    <div>单文件组件：Child2</div>
</template>

<script>
    export default {

    }
</script>

<style scoped>

</style>

3）修改App.vue文件的内容。

<template>
    <!-- 组件模板内容 -->
    <div>
        单文件组件：{{ content }}
        <!-- 使用导入的单文件组件 -->
        <Child1></Child1>
        <Child2></Child2>
    </div>
</template>

<script>
    // 导入其他单文件组件
    import Child1 from './components/Child1.vue'
    import Child2 from './components/Child2.vue'

    // js内容
    export default {
        data: function () {
            return {
                content: 'hello world!'
            }
        },
        // 将其他单文件组件添加到当前组件中
        components: {
            'Child1': Child1,
            'Child2': Child2
        }
    }
</script>

<style scoped>
   /* css样式 */
</style>

4）在终端执行命令将单文件组件进行打包。

npm run build

2. 多组件路由使用

使用路由形式将多个单文件组件组合在一起。

1）将meiduo项目拷贝一份，修改其中的App.vue文件的内容。

<template>
    <!-- 组件模板内容 -->
    <div>
        单文件组件：{{ content }}
    </div>
</template>

<script>
    // js内容
    export default {
        data: function () {
            return {
                content: 'hello world!'
            }
        }
    }
</script>

<style scoped>
   /* css样式 */
</style>

2）在meiduo项目目录下创建路由目录router，并在router目录下创建router.js路由文件。

mkdir router
touch router.js

3）编写路由文件router.js。

import Vue from 'vue'
// 导入路由插件
import Router from 'vue-router'

import Child1 from '../components/Child1.vue'
import Child2 from '../components/Child2.vue'


// 1. 在vue中使用路由插件
Vue.use(Router);

// 2. 定义路由规则
export default new VueRouter({
    routes: [
        {
            path: '/',
            component: Child1
        },
        {
            path: '/child2',
            component: Child2
        }
    ]
})

4）在main.js中使用路由。

import Vue from 'vue'
import App from './App.vue'

// 导入定义好的路由
import router from './router/router.js'


new Vue({
    el: "#app",
    // 使用路由
    router: router,
    render: function (create) {
        return create(App);
    }
});

5）在App.vue中指定路由标签。

<template>
    <!-- 组件模板内容 -->
    <div>
        单文件组件：{{ content }}
        <!-- 路由组件占位符，根据路由规则加载对应的组件内容 -->
        <router-view></router-view>
    </div>
</template>

6）在终端执行命令启动前端服务器。

./node_modules/.bin/webpack-dev-server

注：该服务开启之后，如果修改了项目中的文件，会自动重新进行build操作。
